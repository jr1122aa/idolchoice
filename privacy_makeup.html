개인정보처리방침

Makeup Camera(이하 “앱”)은 이용자의 개인정보 보호를 중요하게 생각하며, 관련 법령을 준수합니다. 본 앱은 회원가입 기능을 제공하지 않으며, 개인을 직접 식별할 수 있는 정보를 최소한으로만 처리합니다.

1. 수집하는 개인정보

앱은 회원가입이 없으며, 아래 항목 외의 개인정보는 수집하지 않습니다.

① 기기 익명 식별값 기록

앱은 서비스 품질 향상 및 오류 해결을 위해 다음과 같은 기록을 서버에 저장합니다.

  • 익명 식별값(기기 고유식별번호가 아닌, 앱 내부에서 생성된 랜덤 ID)

이 정보만으로는 개인을 식별할 수 없습니다.

② 광고를 위한 기기 정보 (AdMob)

앱은 Google AdMob 광고를 사용합니다. AdMob은 광고 제공을 위해 다음과 같은 정보를 자동 수집할 수 있습니다.

  • 광고 ID (AAID/IDFA)
  • 대략적인 위치 정보(도시 단위)
  • 앱 사용 정보

이 정보는 Google이 관리하며, 개발자는 직접 저장하지 않습니다.
자세한 내용은 Google 광고 정책을 참고하세요.

2. 수집 방법

  • 이미지 수정 기능을 사용할 때 서버로 자동 전송되어 저장
  • AdMob SDK를 통해 Google이 자동 수집

3. 개인정보의 이용 목적

  • AI 이미지 수정 기능 제공
  • 서비스 품질 개선 및 오류 분석
  • 비식별 데이터 기반 통계 생성
  • 광고 제공 및 광고 성능 분석(Google AdMob)

4. 개인정보 보관 기간

  • 이미지 수정 기록 : 즉시 삭제
  • 이미지 수정 데이터 : 즉시 삭제
  • AdMob 광고 데이터 : Google의 정책에 따름

5. 개인정보 제3자 제공

앱은 직접적으로 개인정보를 제3자에게 제공하지 않습니다. 단, 광고 제공 과정에서 Google AdMob이 정보를 자동 수집할 수 있습니다.

6. 이용자의 권리

앱은 개인 식별 정보를 수집하지 않기 때문에, 열람/정정/삭제 요청 시 서버에 저장된 생성 기록 삭제가 가능합니다.
이메일을 통해 요청해 주세요.
